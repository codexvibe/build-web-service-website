# ProServices - Agence Digitale Premium

Plateforme moderne de services digitaux (Sites Vitrines, E-commerce, Apps, SEO) développée avec Next.js 15, Tailwind CSS 4 et Supabase.

## 🚀 Fonctionnalités
- **Design Premium** : Interface sombre, moderne et ultra-réactive.
- **Multilingue** : Support complet FR / EN / AR avec gestion RTL.
- **Dashboard Admin** : Gestion des leads, statistiques et administration d'équipe.
- **Formulaire Dynamique** : Calcul automatique des devis selon les besoins clients.

## 🛠️ Stack Technique
- **Frontend** : Next.js 15 (App Router), React 19.
- **Style** : Tailwind CSS v4, Framer Motion.
- **Backend** : Supabase (Database & Auth logic).
- **Hébergement** : Optimisé pour Vercel.

## 📦 Installation
```bash
npm install
npm run dev
```

## ⚙️ Configuration
Créez un fichier `.env.local` :
```env
NEXT_PUBLIC_SUPABASE_URL=votre_url
NEXT_PUBLIC_SUPABASE_ANON_KEY=votre_cle_anon
ADMIN_PASSCODE=votre_code_admin
SESSION_SECRET=votre_secret_jwt
```
